📊 Men's T-Shirt Data Analysis Project
📌 Overview

This project focuses on analyzing men's t-shirt data across different brands to derive meaningful insights related to pricing, ratings, discounts, and product trends. The analysis is performed using modern data tools including Power BI and Azure SQL.

🛠️ Tech Stack

Data Visualization: Power BI

Database: Azure SQL Server

Data Processing: SQL, Power Query

Version Control: GitHub

📂 Dataset Description

The dataset contains information about men's t-shirts from multiple brands, including:

Brand Name

Product Name

Price

Discount

Ratings

Reviews

Category / Type

🎯 Objectives

Compare pricing across different brands

Analyze discount patterns and offers

Identify top-rated brands and products

Understand customer preferences using ratings and reviews

🔄 Data Workflow

Data collected and stored in Azure SQL Server

Data cleaning and transformation using Power Query

Data modeling and relationships created in Power BI

Interactive dashboards built for insights

📊 Key Insights

Identified brands with highest average ratings

Found pricing trends across categories

Highlighted products with highest discounts

Compared brand performance based on customer reviews

📈 Dashboard Features

Brand-wise performance analysis

Price vs Rating comparison

Discount distribution visualization

Interactive filters for better exploration

🚀 How to Use

Clone the repository

Open Power BI file (.pbix)

Connect to Azure SQL (if required)

Refresh data to view updated dashboards
